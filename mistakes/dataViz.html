Choropleth Map // d3-geo, d3-scale, TopoJSON, CSV

Created a map visualizing the number of drug overdose deaths in 2015 per 100,000 people based on every county in the U.S.

Took death rate data from a CSV file, geospatial data from a TopoJSON file generated using QGIS, and color scales from d3 to create a gradient depicting the various overdose rates.

Bubble Chart // d3-force, d3-scale, CSV

Created a bubble chart visualizing the average GDP per country by joining two datasets based on abbreviated country codes.

d3-force was used to create a force layout that spread from the center and each bubble's size and color was made to correspond with the average GDP of the specified country.

A tooltip was created to display the full country name and the average GDP next to the cursor when the cursor hovers over a country's bubble.

Transitioning Scatterplot // d3-scale, CSV

Created a transitioning scatterplot given a dataset of various car specifications (MPG, cylinders, displacement, horsepower, weight, acceleration, year, origin, and name).

Selected variables (acceleration and displacement) for each axis and changed the y-axis variable to displacement after 3 seconds and the x-axis variable to weight after 6 seconds.

For each transition, the change in the x-scale, y-scale, axis labels, and the dots were animated using d3.transition() and the color of each dot was assigned after every update based on the cars' country of origin.

Interactive Line Chart // d3-selection, d3-array, JSON

Created a line chart depicting the predicted height in feet of tides in San Francisco, CA.

Parsed the date, time, and prediction to readable formats and created a filled line chart to depict the actual rise and fall of tides.

A tooltip was created to display the date, time, and height of the tides in an empty white space based on where on the x-axis the cursor was located.
